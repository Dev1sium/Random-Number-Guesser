# Random-Number-Guesser

Random number guessing game, where the user gusses a number between 1 and 100.

Has 3 difficulties (Easy, Medium, Hard).
Easy 	- 	Program will not lie.
Medium 	- 	Program will lie 25% of the time.
Hard 	- 	Program will lie 50% of the time.

Has an editable max attempts, set to a default of 10.

Scoring based on:
Excellent performance 	-	within 2 attempts.
Good performance		- 	within 2 attempts to 49%.
Okay performance	 	-	within 50-69% of max attempts.
Poor performance 		- 	within 70% to 2 less of max attempts.
Bad performance 		- 	when max attempts needed.
